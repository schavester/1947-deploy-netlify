You can use the Tile custom post type to create: button, map, gallery slider, image, and text (with shortcodes support). The Tile posts are displayed using [ tile ] shortcode. Home Category<\/h2>\n

The following shortcode displays 9 posts from homepage category.<\/p>\n

[tile category=”homepage” limit=”9″]<\/p>\n

Below is the social category.<\/p>\n
